## 🌐 الرؤية التقنية (The Sovereign Vision)
تم تحويل NEXUM إلى نظام تشغيل وكلاء (Agentic OS) فائق السرعة، يعتمد على **Gemini 3.1 Flash-Lite** كمحرك عصبي مركزي. هذا الإصدار يركز على زمن الاستجابة المنخفض جداً (Low-Latency) والكفاءة العالية في تنفيذ المهام المعقدة والمتعددة الأنماط (Text, Image, Audio, PDF).

---

## 🚀 المحركات العصبية والتقنية

### 🧠 محرك Gemini 3.1 Flash-Lite (Powered by google-genai)
تم الانتقال بالكامل إلى المكتبة الرسمية الجديدة `google-genai` لتحقيق:
- **سرعة خاطفة**: مثالي للوكلاء الذين يتطلبون استجابة فورية.
- **سياق ضخم**: دعم حتى **1,048,576 Token** في الطلب الواحد.
- **تفكير منطقي (Thinking Mode)**: قدرة عالية على الاستنتاج قبل إنتاج المخرجات.
- **تعدد الأنماط الحقيقي**: استيعاب مباشر للملفات الصوتية، الصور، الفيديوهات، وملفات PDF المعقدة.

### ⚙️ نظام التنفيذ السيادي (Execution Engine)
نظام تخطيط جراحي يقوم بتحويل أهدافك إلى خطوات JSON قابلة للتنفيذ:
- **تحليل الملفات**: قراءة وتعديل الأكواد البرمجية بدقة الأسطر.
- **إدارة الوكلاء**: إنشاء وإدارة Sub-Bots مستقلة.
- **الملاحة الجراحية**: محرك بحث داخلي (Grep) وملاحة هرمية (Tree).

## 🛠️ دليل أوامر التحكم (v3.5.0)

| الأمر | الوصف |
|-------|-------|
| `! [الأمر]` | تنفيذ مباشر عبر **Execution Engine** (تخطيط + تطبيق) |
| `/workspace` | عرض منطقة العمل والملفات النشطة |
| `/tree` | رؤية الهيكل المعماري للمشروع بالتقسيم الهرمي |
| `/cat [file]` | قراءة جراحية لمحتوى أي ملف مع تحديد السطور |
| `/find [query]` | البحث الذكي في المحتوى والأسماء (Pattern Matching) |
| `/stat [path]` | معلومات تقنية شاملة عن الملفات والصلاحيات |
| `/bots` | إدارة شبكة الوكلاء الذكيين ونبض النظام |
