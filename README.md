# 🔱 NEXUM PRIME — Sovereign Agentic Development Infrastructure
> **Autonomous Software Civilization Layer** — ليس مجرد مساعد ذكي، ولكنه بيئة مستقلة بالكامل لوكلاء الذكاء الاصطناعي لتخطيط، بناء، واختبار، ونشر الأنظمة البرمجية.

---

## 🌌 رؤية النظام (The Vision)
**NEXUM PRIME** يتجاوز مفهوم "البوت الذكي" ليقدم **Agentic OS**. هو مصنع وكلاء يقوم بتوزيع المهام، كتابة الأكواد، إنشاء واجهات المستخدم (React/Vite)، إدارة الحاويات (DockerSandbox)، أتمتة مستودعات الأكواد (GitHub Actions)، والتطوير التلقائي. 

الفكرة الأساسية: **"الوكيل يبني الوكيل، والنظام يتطور ذاتياً."**

---

## 🏗️ الهيكلية التقنية (Sovereign Meta-Architecture)
بُني النظام على 5 محركات أساسية:
1. 🧠 **Task Orchestrator**: المايسترو المركزي لإدارة وتوزيع المهام.
2. 🧬 **Agent Registry**: سجل الصلاحيات والقدرات المبني على الكفاءات.
3. 🏭 **Agent Factory**: مصنع لتوليد وكلاء متخصصين ديناميكياً.
4. 🕸️ **Protocol Compiler**: محول الأوامر البشرية إلى مسارات عمل (Workflows).
5. 🛡️ **Sandbox Runtime**: تنفيذ آمن ومعزول تماماً داخل Docker.

---

## 📱 Mobile-First Sovereign Control Plane
يتمتع النظام بـ واجهة تحكم هجينة متطورة:
- **Layer 1: Quick Controls (Telegram Chat)**: أزرار مضمنة لاتخاذ قرارات سريعة واستدعاء لوحة التحكم السيادية.
- **Layer 2: Telegram Mini App Dashboard**: تطبيق ويب مصغر (WebApp) مبني بـ Tailwind و Alpine.js يوفر (Agent Control Center, Sandbox Runtime Monitor, Command Palette).
- **Layer 3: Live Event Stream**: نقل بيانات استهلاك הסيرفر في الوقت الفعلي (CPU, RAM) من البايثون للـ WebApp بشكل حي عبر (SSE).

## 🚀 دليل التشغيل السريع (Quick Start)

### 1️⃣ تثبيت المكتبات (نظام Cloud Shell / Linux)
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2️⃣ التشغيل الحقيقي (تفعيل الواجهة الهجينة)
النظام يعتمد الآن على محركين معاً:
```bash
# 1. شغل البوت السيادي (NEXUM CORE) في تيرمينال
python main.py

# 2. افتح تيرمينال آخر وشغل خادم الربط اللحظي (API Server)
python webapp/api_server.py
```
*(ملاحظة: لكي يعمل الـ WebApp داخل التلجرام، يجب توفير رابط HTTPS وتحديده كمتغير `WEBAPP_URL` داخل ملف `.env`)*
