## 🌌 رؤية النظام (The Vision)
**NEXUM PRIME** يتجاوز مفهوم "البوت الذكي" ليقدم **Agentic OS**. هو مصنع وكلاء يقوم بتوزيع المهام، كتابة الأكواد، إنشاء واجهات المستخدم (React/Vite)، إدارة الحاويات (DockerSandbox)، أتمتة مستودعات الأكواد (GitHub Actions)، والتطوير التلقائي. 

الفكرة الأساسية: **"الوكيل يبني الوكيل، والنظام يتطور ذاتياً."**

---

## 🏗️ الهيكلية التقنية (Sovereign Meta-Architecture)
بُني النظام على 5 محركات أساسية:
1. 🧠 **Task Orchestrator**: المايسترو المركزي لإدارة وتوزيع المهام.
2. 🧬 **Agent Registry**: سجل الصلاحيات والقدرات المبني على الكفاءات.
3. 🏭 **Agent Factory**: مصنع لتوليد وكلاء متخصصين ديناميكياً.
4. 🕸️ **Protocol Compiler**: محول الأوامر البشرية إلى مسارات عمل (Workflows).
5. 🛡️ **Sandbox Runtime**: تنفيذ آمن ومعزول تماماً داخل Docker.

## 🚀 دليل التشغيل السريع (Quick Start)

### 1️⃣ تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2️⃣ تفعيل البيئة الافتراضية (Virtual Environment)
يختلف الأمر حسب نظام التشغيل الخاص بك:

*   **على Linux / Google Cloud Shell:**
    ```bash
    source venv/bin/activate
    ```
*   **على Windows (PowerShell):**
    ```powershell
    .\venv\Scripts\activate
    ```

### 3️⃣ تشغيل النظام
يجب تشغيل البوت الأساسي أولاً، ثم خادم الـ API لوحة التحكم:

```bash
# تشغيل البوت (NEXUM CORE)
python main.py

# تشغيل خادم الـ WebApp (في نافذة منفصلة)
python webapp/api_server.py
```
